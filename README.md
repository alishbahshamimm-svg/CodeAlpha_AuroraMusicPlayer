# Aurora Music Player

**Internship Project — Frontend Development**

- **Student:** Alishbah
- **Company:** Code Alpha
- **Project Type:** Frontend UI / Music player demo

## Project Overview

Aurora Music Player is a dark-themed web UI built with HTML, CSS, and vanilla JavaScript.
It simulates a modern music player interface with a sidebar menu, playback controls, song details, album art, and animated UI elements.

## Key Features

- Responsive player layout with sidebar navigation and control panel
- Theme toggle support (dark/light mode)
- Loading screen and toast notifications
- Music visualization bar and animated album display
- Playback controls including play/pause, previous/next, shuffle, repeat, mute, and volume slider
- Favorite and playlist UI elements
- Local storage support for player state persistence

## Files

- `files/index.html` — main HTML structure for the player UI
- `files/style.css` — styling and layout, including glassmorphism and animations
- `files/script.js` — JavaScript logic for playback simulation, song handling, UI interactions, and state management

## How to Run

1. Open `files/index.html` in a web browser.
2. The player UI will load and show the simulated music experience.

## Notes

- The project uses placeholder album artwork from `picsum.photos`.
- Audio playback is simulated; real music file URLs can be added to the `PLAYLIST` data in `files/script.js`.
- This work is part of an internship project for Code Alpha.
